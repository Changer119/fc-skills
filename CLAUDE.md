# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

`fc-skills` 是一个 Claude Code 技能（Skills）集合仓库，用于存放和管理自定义的 Claude Code slash command 技能。

## 仓库结构

本仓库目前处于初始化阶段。随着技能的添加，结构将逐步完善。

## 技能开发规范

每个技能应包含一个 Markdown 文件，定义技能的触发条件、执行逻辑和提示词内容。具体格式请参考 Claude Code 技能文档。
